# Machine Learning Basics 
<h2>Learning Jupyter Notebook</h2>

<h1>Applied Deep Learning with TensorFlow — Practice Notebook</h1>

<p>
This repository contains a Jupyter Notebook for practicing core TensorFlow concepts and basic deep learning models.
The notebook is based on the assigned task: create a <code>.ipynb</code>, run the code, and ensure everything works.
</p>

<h2>Contents</h2>
<ul>
  <li>TensorFlow tensors (shapes, dtypes, casting, indexing, broadcasting)</li>
  <li>Matrix multiplication and aggregation operations</li>
  <li>One-hot encoding and NumPy interoperability</li>
  <li>Regression model (y = x + 10)</li>
  <li>Binary classification (circles dataset)</li>
  <li>Multiclass classification (Fashion-MNIST)</li>
  <li>CNN binary classification (Fashion-MNIST: class 7 vs 9)</li>
  <li>Small CNN (CIFAR-10 subset: classes 0–2)</li>
</ul>

<h2>Requirements</h2>
<ul>
  <li>Python 3.10 / 3.11</li>
  <li>TensorFlow</li>
  <li>NumPy</li>
  <li>Matplotlib</li>
  <li>scikit-learn</li>
  <li>Jupyter</li>
</ul>

<h2>Setup</h2>
<ul>
  <li>Create and activate a virtual environment</li>
  <li>Install dependencies</li>
  <li>Run the notebook in JupyterLab</li>
</ul>

<h3>Install dependencies</h3>
<pre>
pip install tensorflow numpy matplotlib scikit-learn jupyter ipykernel
</pre>

<h2>Run</h2>
<pre>
jupyter lab
</pre>

<h2>Notes</h2>
<ul>
  <li>Fashion-MNIST and CIFAR-10 are downloaded automatically on first run.</li>
  <li>If dataset download fails due to SSL on macOS, ensure Python certificates are installed and SSL_CERT_FILE is set.</li>
</ul>
